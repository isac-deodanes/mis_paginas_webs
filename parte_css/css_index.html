Introducción

CSS3 (Cascading Style Sheets 3) es la última versión del lenguaje de hojas de estilo en cascada utilizado para definir los estilos y la apariencia de las páginas web. CSS se ha convertido en una herramienta indispensable para los diseñadores y desarrolladores de hoy en día.
Ha transformado la apariencia y la experiencia de navegación en la web.
A raíz de todo lo anterior, te preguntaras: ¿Qué es CSS3?

¿Qué es CSS3?

CSS3 es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web, e interfaces de usuario escritas en HTML o XHTML; el lenguaje puede ser aplicado a cualquier documento XML, incluyendo XHTML, SVG, XUL, RSS, etcétera. Junto con HTML y JavaScript, CSS es una tecnología usada por muchos sitios web para crear páginas visualmente atractivas, interfaces de usuario para aplicaciones web y GUIs para muchas aplicaciones móviles (como Firefox OS).

Generalidades de CSS3

CSS3 ue lanzado en 1999 y ha ido actualizándose a lo largo de los años para incluir nuevas funcionalidades y mejoras en la presentación de las páginas web1. Algunas de las características más importantes de CSS3 incluyen:

Propiedades avanzadas de texto

CSS3 ofrece propiedades adicionales para personalizar el aspecto del texto, como text-shadow para agregar sombras al texto y text-overflow para manejar el desbordamiento de texto en contenedores.

Transformaciones y transiciones

Puedes aplicar transformaciones (como rotación, escalado y traslación) a elementos HTML utilizando CSS3. Además, las transiciones permiten crear efectos suaves al cambiar propiedades CSS.

Gradientes y transparencias

CSS3 introduce la posibilidad de aplicar gradientes de color a fondos y bordes. Además, puedes controlar la transparencia de los elementos utilizando propiedades como opacity.

Flexbox y Grid

CSS3 introduce los modelos de diseño flexbox y grid, que simplifican la creación de diseños complejos y flexibles.

¿Como usar CSS3 en mis proyectos?

Como atributo

Podemos utilizar el atributo "Style", para estilizar una etiqueta en específico colocando como valor la propiedad a la que queremos darle estilo, sin embargo, esta es una forma muy poco práctica para estilizar proyectos con CSS3.

Como etiqueta

Utilizando la etiqueta "Style" dentro del documento HTML en el que estamos trabajando, podemos seleccionar todas las etiquetas de un mismo tipo, podemos hacerlo por clases, o por medio de un id asignado, y modificar las propiedades que deseemos de las etiquetas seleccionadas

CSS Externo

Aunque sin duda, la manera más práctica para aplicar CSS en nuestros proyectos, es crear un archivo .css, y enlazarlo a nuestro archivo HTML, y el proceso es el mismo que con la etiqueta Style, seleccionamos la etiqueta a la que queremos darle estilo, y modificamos sus propiedades, sin embargo, el crear un archivo .css y enlazarlo es la manera más práctica de utilizar CSS, además que hace que nuestro código sea mucho más legible.
