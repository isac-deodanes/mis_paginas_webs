EMMET

Introducción

Emmet es un conjunto de herramientas y técnicas que aceleran el flujo de trabajo de los desarrolladores web al escribir código HTML y CSS. Originalmente conocido como Zen Coding, Emmet permite escribir abreviaturas y expandirlas automáticamente en código completo.

Para aprovechar Emmet, de preferencia, es recomendable utilizar el editor de código: Visual Studio Code, que ya tiene integrada la extensión de Emmet

Link: Tutorial de instalación de VS Code

En lugar de escribir código HTML o CSS completo, puedes utilizar abreviaturas para representar estructuras comunes.
Por ejemplo, en lugar de escribir <div class="container">, puedes simplemente escribir .container y presionar la tecla Tab para expandirlo automáticamente.

Es asi de simple, con Emmet, puedes olvidarte de tener que escribir los signos menor y mayor que "< >", para cada etiqueta.

Emmet HTML

Para aprovechar Emmet en tu código de HTML, lo primero que debes saber, es que no es necesario que escribas los signos menor y mayor que "< >", y tampoco es necesario que escribas la etiqueta de cierre para cada etiqueta, basta con que escribas el nombre de la etiqueta y presiones la tecla tab o la tecla enter para expandir la etiqueta automaticamente.

Selectores

Con Emmet, también podemos agregar selectores como clases y id's automaticamente a las etiquedas, lo hacemos colocando el nombre de la etiqueta, seguido de un punto ".", o de una almoadilla "#", seguido del nombre que se desea asignar al selector.
Ejemplo:

div.container, con un "." designamos una clase a la etiqueta

Resultado:

div#container, con el signo "#", designamos un id a la etiqueta

Resultado:

Estructuras complejas

Gracias a Emmet, podemos escribir porciones grandes de código en una sola linea, que se expande y crea una estrutura compleja, para ello utilizamos el signo mayor que ">", que nos permite crear una etiqueta dentro de otra, esto es muy práctico al momento de realizar la maquetación de un sitio.

Ejemplo: div > p

Resultado:

También, si necesitamos escribir varias veces una misma etiqueta, Emmet nos ahorra ese trabajo,solo necesitamos escribir el nombre de la etiqueta, seguido de un asterisco "*", seguido de la cantidad de veces que queremos que se repita la etiqueta.

Ejemplo:

Con lo aprendido hasta el momento, podemos crear estructuras complejas en una sola linea de código, por ejemplo, podemos crear un header que funcione como menú de navegación.

Ejemplo:

header > img + nav > a * 5

Resultado:

El signo "+" se utiliza para crear etiquetas diferentes después de una etiqueta, que vendrian siendo etiquetas hermanas

Enumeraciones y contenido

Para ingresar contenido de texto usando Emmet, se hace usando los signos de las llaves "{}", se pone el nombre de la etiqueta, seguido de las llaves, y dentro de estas, el contenido de texto que deseemos ingresar en la etiqueta, o que se muestre en el navegador.

Ejemplo:

p{Hola mundo}

Resultado:

También podemos enumerar, usando el signo de dólar "$", podemos hacerlo con el contenido de las etiquetas, con las clases, o con los id´s

Ejemplo:

div.container-$*6

Abreviaturas Emmet

A continuación se presentan una lista de abreviaturas en Emmet, y su resultado

Abreviatura Emmet Resultado
p <p> </p>
h1 <h1> </h1>
tab <table> </table>
form <form action=""> </form>
div <div> </div>
header <header> </header>
footer <footer> </footer>
section <section> </section>
img <img src="" alt="">
tr <tr> </tr>
td <td> </td>

Con Emmet en HTML, por lo general basta con escribir el nombre de la etiqueta, y esta se expandirá automaticamente, incluyendo la etiqueta de cierre.

Es importante aprender Emmet, para optimizar los tiempos al momento de escribir HTML y CSS, con Emmet nos ahorramos el estar escribiendo una gran estructura cáracter por cáractar, con este plugin podemos hacerlo en una sola linea de código.

Emmet CSS

También podemos escribir CSS usando, en este caso, está enfocado, principalmente en los atributos, Emmet cuenta con abreviaciones, para modificar los atributos de las etiquetas de manera más rápida y práctica.
