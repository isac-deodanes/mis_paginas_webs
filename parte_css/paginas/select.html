Selectores

Introducción

En CSS los selectores se utilizan para delimitar los elementos HTML de nuestra página web a los que queremos aplicar estilo. Hay una amplia variedad de selectores CSS, lo que permite una gran precisión a la hora de seleccionar elementos a los que aplicar estilo. En este artículo y sus subartículos veremos con detalle todos los tipos y el modo como funcionan.

¿Qué es un selector?

Un selector CSS es la primera parte de una regla CSS. Es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS. El elemento o los elementos seleccionados por el selector se denominan sujeto del selector.

La etiqueta h1 funciona como selector, en el caso de que usemos como selector el nombre de una etiqueta, eso quiere decir que todos los atributos que modifiquemos, se aplicaran a todas las etiquetas de ese tipo, en este caso a todas las etiquetas h1.

También existen otros tipo de selectores, como id´s y clases

Pero antes, veamos como agrupar selectores, podemos hacerlo usando una coma entre los selectores, para aplicar los mismos estilos a diferentes etiquetas, o selectores

Selectores de tipo, de clase, y de Id

Los selectores de tipo Id y clase, se utilizan para seleccionar etiquetas más especificas, a las que previamente, en el archivo HTML, se les asignó un Id o una clase.

Clase: Las clases se utilizan para seleccionar un conjunto de etiquetas, que pueden ser del mismo tipo, o diferentes, para posteriormente aplicarle los mismos estilos.

En el documento HTML, se ven de la siguiente manera: <div class="container"></div>

Y en el documento CSS se ven de la siguiente manera: .container

Id: Se le asigna unicamente a un elemento, al que se le quiere cambiar su estilo en CSS.

En el documento HTML, se ven de la siguiente manera: <div id=container"></div>

Y en el documento CSS se ven de la siguiente manera: #container
