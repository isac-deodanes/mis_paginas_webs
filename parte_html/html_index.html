Introducción

HTML5 es la última versión del lenguaje de marcado HTML (Hypertext Markup Language).
Es un lenguaje de marcas que nos permite crear páginas web.
HTML está presente en la totalidad de sitios web que podemos encontrar en la web, pues este lenguaje de marcado constituye la espina dorsal de todo sitio web.

¿Qué es HTML5?

HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros…

Generalidades de HTML5

En primer lugar, es importante aclarar que HTML5 no es un lenguaje de programación, sino un lenguaje de marcado que define cómo se presenta y se organiza el contenido en una página web.

Estructurar el contenido

HTML5 te permite definir la estructura de una página web mediante etiquetas como <header>, <nav>, <section>, <article>, <footer>, etc.

Puedes organizar tu contenido de manera semántica y significativa.

Incorporar multimedia

HTML5 permite la inclusión de elementos multimedia como audio y vídeo directamente en las páginas web sin necesidad de plugins.

Puedes insertar imágenes, gráficos, animaciones y más.

Formularios interactivos

HTML5 ofrece etiquetas para crear formularios interactivos con campos de entrada, botones, casillas de verificación, etc.
Puedes recopilar datos de los usuarios y enviarlos al servidor.

En resumen, HTML5 es esencial para el desarrollo web moderno y ofrece muchas ventajas en comparación con versiones anteriores.

Sintaxis

La sintaxis de HTML (Hypertext Markup Language) es fundamental para crear páginas web. HTML se basa en etiquetas o marcas que definen la estructura y el contenido de un documento. Aquí tienes algunos conceptos clave sobre la sintaxis de HTML:

Se definen mediante símbolos de “menor que” (<) y “mayor que” (>).

Existen etiquetas de apertura y cierre, como <etiqueta> y </etiqueta>.

Las etiquetas encierran el contenido y definen su función en la página.

Ejemplos de etiquetas en HTML

<p>: Define un párrafo.
<h1>, <h2>, …, <h6>: Encabezados de diferentes niveles.
<a>: Crea enlaces.
<img>: Inserta imágenes.
<ul> y <li>: Crean listas no ordenadas
