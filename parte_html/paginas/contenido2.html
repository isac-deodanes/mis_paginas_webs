La etiqueta <audio> para insertar sonido dentro de una página web, se recomienda usar archivos con formato *.mp3.por ser mas livianos y permitir que la pagina web cargue mas rápido

VIDEOS DE YOUTUBE

Cuando se desea presentar en una página web un video remoto, como por ejemplo los videos de YouTube, es necesario utilizar la etiqueta <iframe> ya que se trata de la incrustación de un objeto externo dentro de nuestra página web.

PROCESO..

  1. Abrimos youtube.
  2. Buscamos el video que necesitamos.
  3. Ponemos el puntero del mouse encima del video y presionamos clic derecho.
  4. Seleccionamos la opción < > Copiar código de inserción.
  5. Acontinuación pegamos el código en nuestra pagina web.

ATRIBUTO autoplay.

Sirve para reproducir el vídeo automáticamente cuando este se haya cargado sin consultar con el usuario. No verdad, trata de usarlo siempre y cuando consideres necesario.
